```
내 가게를 소개하는 1페이지 웹사이트 index.html을 만들어줘.
가게 정보:
- 상호명: [모닝브루]
- 업종: [동네 카페]
- 위치: [서울 망원동 골목 끝, 망원역 2번 출구 도보 6분]
- 영업시간: [화~일 08:00~20:00, 월요일 휴무]
- 대표 상품/메뉴: [핸드드립 커피 5,500원, 오늘의 원두 4,500원, 수제 스콘 3,500원, 계절 과일 에이드 6,000원]
- 특징: [매일 아침 직접 내리는 핸드드립, 당일 구운 수제 스콘, 혼자 와도 편한 1인 좌석]
- 연락처: [인스타그램 @morningbrew_official, 전화 02-123-4567]
페이지 구성:
1. 첫 화면에는 가게 이름과 한 줄 소개를 크게 보여줘
2. 대표 상품/메뉴를 카드 형태로 보여줘
3. 위치와 영업시간을 쉽게 찾을 수 있게 해줘
4. 연락 또는 예약 버튼을 눈에 띄게 넣어줘
5. 모바일 화면에서 먼저 보기 좋게 만들어줘
디자인:
- 따뜻하고 신뢰감 있는 분위기
- 배경은 밝게
- 버튼은 잘 보이는 색상으로
한글이 깨지지 않게 <meta charset="UTF-8">가 있는지 확인해줘.
```
